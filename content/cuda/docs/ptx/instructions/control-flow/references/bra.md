# PTX Instruction Topic: bra

`bra` is a fundamental PTX branch instruction that supports predicate-controlled conditional branching.

## Official Description
- Documentation section: Control Flow Instructions: `bra`
- Commonly used together with predicates generated by `setp`

## Key Constraints
- Conditional branching depends on the result of a predicate register.
- Branch divergence can affect warp execution efficiency.
- The target label must be within a valid control-flow range.

## Example (PTX Style)

```ptx
@p bra L_true;
bra L_end;
```
